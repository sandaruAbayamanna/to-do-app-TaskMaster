Task Master TO-DO app:

1. Download and unzip the project file attached

2. Launch Android Studio on your machine.

2. Goto files-> open and select the path where you unzip the project file

4. Build the Project:
Click on the "Build" menu in Android Studio and select "Make Project" to compile the code.

5. Run the App:
Click on the "Run" menu in Android Studio and select "Run 'app'" or use the green play button. This will build and deploy the app to an emulator or a connected device.

6. Test the To-Do List App:
Once the app is running, test the functionalities that implemented, such as adding, editing, and deleting tasks by interacting with the app. Also, the tasks can be marked as completed as well.
